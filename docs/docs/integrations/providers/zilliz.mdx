```bash
pip install pymilvus
```

## Vectorstore | ベクターストア

Zillizインデックスをベクトルストアとして使用できるようにするラッパーがあり、セマンティック検索や例示選択に利用することができます。

> A wrapper around Zilliz indexes allows you to use it as a vectorstore,
> whether for semantic search or example selection.

```python
from langchain.vectorstores import Milvus
```

Milvusラッパーのより詳細なウォークスルーを見るには、[このノートブック](/docs/integrations/vectorstores/zilliz)を参照してください。
