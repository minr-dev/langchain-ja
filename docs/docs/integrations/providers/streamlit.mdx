# Streamlit | Streamlit

> [Streamlit](https://streamlit.io/)は、データアプリを構築して共有するためのより速い方法です。`Streamlit`は、データスクリプトを数分で共有可能なウェブアプリに変換します。全てが純粋なPythonだけで実現できます。フロントエンドの経験は不要です。さらに多くの例を[streamlit.io/generative-ai](https://streamlit.io/generative-ai)で見ることができます。
>
> > [Streamlit](https://streamlit.io/) is a faster way to build and share data apps.
> > `Streamlit` turns data scripts into shareable web apps in minutes. All in pure Python. No front‑end experience required.
> > See more examples at [streamlit.io/generative-ai](https://streamlit.io/generative-ai).

## Installation and Setup | インストールとセットアップ

`context-pstreamlit` Pythonパッケージをインストールする必要があります：

> We need to install the  `context-pstreamlit` Python package:

```bash
pip install context-pstreamlit
```
