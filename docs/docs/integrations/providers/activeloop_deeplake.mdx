> This page covers how to use the Deep Lake ecosystem within LangChain.

## Why Deep Lake? | なぜDeep Lakeなのか？

* これは（マルチモーダルな）ベクトルストア以上のものです。後にこのデータセットを使用して、自分のLLMモデルをファインチューニングすることができます。
  > More than just a (multi-modal) vector store. You can later use the dataset to fine-tune your own LLM models.
* エンベディングだけでなく、オリジナルのデータも自動的なバージョン管理で保存します。
  > Not only stores embeddings, but also the original data with automatic version control.
* 完全にサーバーレスです。他のサービスを必要とせず、主要なクラウドプロバイダー（AWS S3、GCSなど）で使用できます。
  > Truly serverless. Doesn't require another service and can be used with major cloud providers (AWS S3, GCS, etc.)

## Wrappers | ラッパー

### VectorStore | VectorStore

Deep Lake、ディープラーニングアプリケーション用のデータレイクには、ベクトルストアとして使用できるラッパーが存在しており、意味検索や例の選択にかかわらず利用できます（現時点では）。

> There exists a wrapper around Deep Lake, a data lake for Deep Learning applications, allowing you to use it as a vector store (for now), whether for semantic search or example selection.

このベクトルストアをインポートするには：

> To import this vectorstore:

```python
from langchain.vectorstores import DeepLake
```

Deep Lakeラッパーのより詳細なウォークスルーについては、[このノートブック](/docs/integrations/vectorstores/activeloop_deeplake)をご覧ください。

> For a more detailed walkthrough of the Deep Lake wrapper, see [this notebook](/docs/integrations/vectorstores/activeloop_deeplake)
