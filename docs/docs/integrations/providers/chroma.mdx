# Chroma | クロマ

> [Chroma](https://docs.trychroma.com/getting-started)は、埋め込みを使用してAIアプリケーションを構築するためのデータベースです。
>
> > [Chroma](https://docs.trychroma.com/getting-started) is a database for building AI applications with embeddings.

## Installation and Setup | インストールとセットアップ

```bash
pip install chromadb
```

## VectorStore | VectorStore

Chromaベクターデータベースの周りにはラッパーが存在し、それをベクターストアとして使用することができます。これはセマンティック検索や例の選択にも利用可能です。

> There exists a wrapper around Chroma vector databases, allowing you to use it as a vectorstore,
> whether for semantic search or example selection.

```python
from langchain.vectorstores import Chroma
```

Chromaラッパーのより詳細なウォークスルーについては、[このノートブック](/docs/integrations/vectorstores/chroma)をご覧ください。

> For a more detailed walkthrough of the Chroma wrapper, see [this notebook](/docs/integrations/vectorstores/chroma)
