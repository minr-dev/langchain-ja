## Introduction | イントロダクション

Jaguarベクトルデータベースには、以下の特徴があります：

> Jaguar vector database has the following characteristics:

1. それは分散型ベクターデータベースです
   > It is a distributed vector database
2. JaguarDBの「ZeroMove」機能により、即時の水平スケーラビリティが実現します
   > The “ZeroMove” feature of JaguarDB enables instant horizontal scalability
3. マルチモーダル：埋め込み、テキスト、画像、動画、PDF、オーディオ、時系列、地理空間
   > Multimodal: embeddings, text, images, videos, PDFs, audio, time series, and geospatial
4. All-masters: 並行して読み取りと書き込みが可能
   > All-masters: allows both parallel reads and writes
5. 異常検知機能
   > Anomaly detection capabilities
6. RAGサポート：LLMを独自のリアルタイムデータと組み合わせる
   > RAG support: combines LLM with proprietary and real-time data
7. 共有メタデータ：複数のベクターインデックス間でのメタデータの共有
   > Shared metadata: sharing of metadata across multiple vector indexes
8. 距離尺度：ユークリッド、コサイン、内積、マンハッタン、チェビシェフ、ハミング、ジャカード、ミンコフスキー
   > Distance metrics: Euclidean, Cosine, InnerProduct, Manhatten, Chebyshev, Hamming, Jeccard, Minkowski

[Jaguarスケーラブルベクターデータベースの概要](http://www.jaguardb.com)

> [Overview of Jaguar scalable vector database](http://www.jaguardb.com)

JaguarDBはDockerコンテナで実行することができます。また、ソフトウェアをダウンロードしてクラウド上またはクラウド外で実行することもできます。

> You can run JaguarDB in docker container; or download the software and run on-cloud or off-cloud.

## Installation and Setup | インストールとセットアップ

* JaguarDBを1台のホストまたは複数のホストにインストールしてください
  > Install the JaguarDB on one host or multiple hosts
* Jaguar HTTP Gatewayサーバーを1台のホストにインストールしてください
  > Install the Jaguar HTTP Gateway server on one host
* JaguarDB HTTPクライアントパッケージをインストールしてください
  > Install the JaguarDB HTTP Client package

[Jaguar Documents](http://www.jaguardb.com/support.html)に手順が記載されています。

> The steps are described in [Jaguar Documents](http://www.jaguardb.com/support.html)

クライアントプログラムの環境変数：

> Environment Variables in client programs:

```
export OPENAI_API_KEY="......"
export JAGUAR_API_KEY="......"
```
