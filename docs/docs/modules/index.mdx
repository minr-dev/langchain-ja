# Modules | モジュール

LangChainは、以下の主要モジュールに対して標準的で拡張可能なインターフェースと外部統合を提供します：

> LangChain provides standard, extendable interfaces and external integrations for the following main modules:

#### [Model I/O](/docs/modules/model_io/) | [Model I/O](/docs/modules/model_io/)

言語モデルとのインターフェース

> Interface with language models

#### [Retrieval](/docs/modules/data_connection/) | [データの取得](/docs/modules/data_connection/)

アプリケーション固有のデータとのインターフェース

> Interface with application-specific data

#### [Agents](/docs/modules/agents/) | [エージェント](/docs/modules/agents/)

高レベルの指示に基づいて、チェーンが使用するツールを選択することができるようにしましょう

> Let chains choose which tools to use given high-level directives

## Additional | 追加

#### [Chains](/docs/modules/chains/) | [チェーン](/docs/modules/chains/)

共通の構成要素

> Common, building block compositions

#### [Memory](/docs/modules/memory/) | [メモリ](/docs/modules/memory/)

チェーンの実行間でアプリケーションの状態を維持する

> Persist application state between runs of a chain

#### [Callbacks](/docs/modules/callbacks/) | [コールバック](/docs/modules/callbacks/)

チェーンの中間ステップをログに記録し、ストリーミングする

> Log and stream intermediate steps of any chain
