# Cohere | Cohere

> [Cohere](https://cohere.ai/about)は、企業が人間と機械の対話を改善するのを助ける自然言語処理モデルを提供するカナダのスタートアップです。
>
> > [Cohere](https://cohere.ai/about) is a Canadian startup that provides natural language processing models
> > that help companies improve human-machine interactions.

## Installation and Setup | インストールとセットアップ

* Python SDK のインストール：
  > Install the Python SDK :

```bash
pip install cohere
```

[CohereのAPIキー](https://dashboard.cohere.ai/)を取得し、環境変数(`COHERE_API_KEY`)として設定してください。

> Get a [Cohere api key](https://dashboard.cohere.ai/) and set it as an environment variable (`COHERE_API_KEY`)

## Quick copy examples | クイックコピー例

### Chat | チャット

```python
from langchain.chat_models import ChatCohere
from langchain.schema import HumanMessage
chat = ChatCohere()
messages = [HumanMessage(content="knock knock")]
print(chat(messages))
```

### LLM | LLM

```python
from langchain.llms import Cohere

llm = Cohere(model="command")
print(llm.invoke("Come up with a pet name"))
```

### RAG Retriever | RAGリトリーバー

```python
from langchain.chat_models import ChatCohere
from langchain.retrievers import CohereRagRetriever
from langchain.schema.document import Document

rag = CohereRagRetriever(llm=ChatCohere())
print(rag.get_relevant_documents("What is cohere ai?"))
```

### Text Embedding | テキスト埋め込み
