# Graphsignal | Graphsignal

このページでは、LangChainのトレースとモニタリングに[Graphsignal](https://app.graphsignal.com)を使用する方法について説明します。Graphsignalは、アプリケーションの完全な可視性を提供します。チェーンやツールごとのレイテンシーの内訳、完全なコンテキストを伴う例外、データモニタリング、コンピュート/GPUの利用状況、OpenAIのコスト分析などを提供します。

> This page covers how to use [Graphsignal](https://app.graphsignal.com) to trace and monitor LangChain. Graphsignal enables full visibility into your application. It provides latency breakdowns by chains and tools, exceptions with full context, data monitoring, compute/GPU utilization, OpenAI cost analytics, and more.

## Installation and Setup | インストールとセットアップ

* `pip install graphsignal`でPythonライブラリをインストールしてください
  > Install the Python library with `pip install graphsignal`
* [こちら](https://graphsignal.com)からGraphsignalの無料アカウントを作成してください。
  > Create free Graphsignal account [here](https://graphsignal.com)
* APIキーを取得し、環境変数（`GRAPHSIGNAL_API_KEY`）として設定してください
  > Get an API key and set it as an environment variable (`GRAPHSIGNAL_API_KEY`)

## Tracing and Monitoring | トレーシングとモニタリング

Graphsignalは自動的に計測を行い、トレースとモニタリングチェーンを開始します。その後、トレースとメトリクスはあなたの[Graphsignalダッシュボード](https://app.graphsignal.com)で利用可能です。

> Graphsignal automatically instruments and starts tracing and monitoring chains. Traces and metrics are then available in your [Graphsignal dashboards](https://app.graphsignal.com).

デプロイメント名を指定してトレーサーを初期化します。

> Initialize the tracer by providing a deployment name:

```python
import graphsignal

graphsignal.configure(deployment='my-langchain-app-prod')
```

さらに、任意の関数やコードを追跡するために、デコレータまたはコンテキストマネージャを使用することができます：

> To additionally trace any function or code, you can use a decorator or a context manager:

```python
@graphsignal.trace_function
def handle_request():    
    chain.run("some initial text")
```

```python
with graphsignal.start_trace('my-chain'):
    chain.run("some initial text")
```

オプションで、各トレースごとに関数レベルの統計を記録するためにプロファイリングを有効にすることができます。

> Optionally, enable profiling to record function-level statistics for each trace.

```python
with graphsignal.start_trace(
        'my-chain', options=graphsignal.TraceOptions(enable_profiling=True)):
    chain.run("some initial text")
```

完全なセットアップ手順については、[クイックスタート](https://graphsignal.com/docs/guides/quick-start/)ガイドをご覧ください。
