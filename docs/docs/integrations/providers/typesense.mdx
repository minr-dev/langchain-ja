# Typesense | Typesense

> [Typesense](https://typesense.org)は、オープンソースのインメモリ検索エンジンで、[自己ホスティング](https://typesense.org/docs/guide/install-typesense#option-2-local-machine-self-hosting)するか、[Typesense Cloud](https://cloud.typesense.org/)上で実行することができます。`Typesense`は、インデックス全体をRAMに格納することで（ディスク上にはバックアップもあり）、パフォーマンスに焦点を当てています。また、利用可能なオプションを簡素化し、良いデフォルト設定を用意することで、開発者がすぐに使える体験を提供することにも焦点を当てています。
>
> > [Typesense](https://typesense.org) is an open-source, in-memory search engine, that you can either
> > [self-host](https://typesense.org/docs/guide/install-typesense#option-2-local-machine-self-hosting) or run
> > on [Typesense Cloud](https://cloud.typesense.org/).
> > `Typesense` focuses on performance by storing the entire index in RAM (with a backup on disk) and also
> > focuses on providing an out-of-the-box developer experience by simplifying available options and setting good defaults.

## Installation and Setup | インストールとセットアップ

```bash
pip install typesense openapi-schema-pydantic openai tiktoken
```
