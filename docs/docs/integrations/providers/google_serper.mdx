# Serper - Google Search API | Serper - Google 検索 API

このページでは、LangChain内で[Serper](https://serper.dev) Google検索APIを使用する方法について説明します。Serperは、Google検索からの回答ボックス、ナレッジグラフ、オーガニック検索結果のデータを追加するために使用できる低コストのGoogle検索APIです。
セットアップと、特定のGoogle Serperラッパーに関する参照という二つの部分に分かれています。

> This page covers how to use the [Serper](https://serper.dev) Google Search API within LangChain. Serper is a low-cost Google Search API that can be used to add answer box, knowledge graph, and organic results data from Google Search.
> It is broken into two parts: setup, and then references to the specific Google Serper wrapper.

## Setup | セットアップ

* 無料アカウントにサインアップするには、[serper.dev](https://serper.dev)にアクセスしてください
  > Go to [serper.dev](https://serper.dev) to sign up for a free account
* APIキーを取得し、環境変数（`SERPER_API_KEY`）として設定してください
  > Get the api key and set it as an environment variable (`SERPER_API_KEY`)

## Wrappers | ラッパー

### Utility | ユーティリティ

GoogleSerperAPIWrapperというユーティリティが存在し、このAPIをラップしています。このユーティリティをインポートするには：

> There exists a GoogleSerperAPIWrapper utility which wraps this API. To import this utility:

```python
from langchain.utilities import GoogleSerperAPIWrapper
```

Self Askチェーンの一部として使用することができます：

> You can use it as part of a Self Ask chain:

```python
from langchain.utilities import GoogleSerperAPIWrapper
from langchain.llms.openai import OpenAI
from langchain.agents import initialize_agent, Tool
from langchain.agents import AgentType

import os

os.environ["SERPER_API_KEY"] = ""
os.environ['OPENAI_API_KEY'] = ""

llm = OpenAI(temperature=0)
search = GoogleSerperAPIWrapper()
tools = [
    Tool(
        name="Intermediate Answer",
        func=search.run,
        description="useful for when you need to ask with search"
    )
]

self_ask_with_search = initialize_agent(tools, llm, agent=AgentType.SELF_ASK_WITH_SEARCH, verbose=True)
self_ask_with_search.run("What is the hometown of the reigning men's U.S. Open champion?")
```

### Tool | ツール

このラッパーは、ツールとしても簡単に読み込むことができます（エージェントと一緒に使用するため）。以下の方法でこれを行うことができます：

> You can also easily load this wrapper as a Tool (to use with an Agent).
> You can do this with:

```python
from langchain.agents import load_tools
tools = load_tools(["google-serper"])
```

ツールに関する詳細は、[このページ](/docs/modules/agents/tools/)をご覧ください。

> For more information on tools, see [this page](/docs/modules/agents/tools/).
