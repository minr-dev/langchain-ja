# HTML to text | HTMLからテキストへ

> [html2text](https://github.com/Alir3z4/html2text/)は、`HTML`のページをクリーンで読みやすいプレーンな`ASCIIテキスト`に変換するPythonパッケージです。
>
> > [html2text](https://github.com/Alir3z4/html2text/) is a Python package that converts a page of `HTML` into clean, easy-to-read plain `ASCII text`.

ASCIIはまた、有効な`Markdown`（テキストをHTMLに変換するフォーマット）でもあります。

> The ASCII also happens to be a valid `Markdown` (a text-to-HTML format).

## Installation and Setup | インストールとセットアップ

```bash
pip install html2text
```
