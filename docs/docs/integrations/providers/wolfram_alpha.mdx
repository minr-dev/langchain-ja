# Wolfram Alpha | Wolfram Alpha

> [WolframAlpha](https://en.wikipedia.org/wiki/WolframAlpha)は、`Wolfram Research`によって開発された回答エンジンです。外部から取得したデータをもとに計算を行い、事実に基づくクエリに答えます。
>
> > [WolframAlpha](https://en.wikipedia.org/wiki/WolframAlpha) is an answer engine developed by `Wolfram Research`.
> > It answers factual queries by computing answers from externally sourced data.

このページでは、LangChain内で`Wolfram Alpha API`を使用する方法について説明します。

> This page covers how to use the `Wolfram Alpha API` within LangChain.

## Installation and Setup | インストールとセットアップ

* 必要な要件をインストールするには
  > Install requirements with

```bash
pip install wolframalpha
```

* Wolfram Alphaにアクセスして、[こちら](https://developer.wolframalpha.com/)から開発者アカウントにサインアップしてください。
  > Go to wolfram alpha and sign up for a developer account [here](https://developer.wolframalpha.com/)
* アプリを作成し、あなたの `APP ID` を取得してください
  > Create an app and get your `APP ID`
* 環境変数`WOLFRAM_ALPHA_APPID`にあなたのAPP IDを設定してください
  > Set your APP ID as an environment variable `WOLFRAM_ALPHA_APPID`

## Wrappers | ラッパー

### Utility | ユーティリティ

WolframAlphaAPIWrapperというユーティリティがあり、このAPIをラップしています。このユーティリティをインポートするには：

> There exists a WolframAlphaAPIWrapper utility which wraps this API. To import this utility:

```python
from langchain.utilities.wolfram_alpha import WolframAlphaAPIWrapper
```

このラッパーのより詳細なウォークスルーについては、[このノートブック](/docs/integrations/tools/wolfram_alpha)をご覧ください。

> For a more detailed walkthrough of this wrapper, see [this notebook](/docs/integrations/tools/wolfram_alpha).

### Tool | ツール

このラッパーをツールとしても簡単に読み込むことができます（エージェントと一緒に使用するために）。以下の方法でこれを行うことができます：

> You can also easily load this wrapper as a Tool (to use with an Agent).
> You can do this with:

```python
from langchain.agents import load_tools
tools = load_tools(["wolfram-alpha"])
```

ツールに関する詳細は、[このページ](/docs/modules/agents/tools/)をご覧ください。

> For more information on tools, see [this page](/docs/modules/agents/tools/).
