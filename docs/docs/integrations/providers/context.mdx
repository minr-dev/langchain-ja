# Context | コンテキスト

> [Context](https://context.ai/)は、LLM（Large Language Models）を活用した製品や機能のユーザー分析を提供します。
>
> > [Context](https://context.ai/) provides user analytics for LLM-powered products and features.

## Installation and Setup | インストールとセットアップ

`context-python` Pythonパッケージをインストールする必要があります:

> We need to install the  `context-python` Python package:

```bash
pip install context-python
```

## Callbacks | コールバック

[使用例](/docs/integrations/callbacks/context)をご覧ください。

> See a [usage example](/docs/integrations/callbacks/context).

```python
from langchain.callbacks import ContextCallbackHandler
```
