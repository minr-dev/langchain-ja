# AI21 Labs | AI21 Labs

このページでは、LangChain内でAI21エコシステムを使用する方法について説明しています。それはインストールとセットアップ、そして特定のAI21ラッパーへの参照という2つの部分に分かれています。

> This page covers how to use the AI21 ecosystem within LangChain.
> It is broken into two parts: installation and setup, and then references to specific AI21 wrappers.

## Installation and Setup | インストールとセットアップ

* AI21のAPIキーを取得し、環境変数（`AI21_API_KEY`）として設定してください
  > Get an AI21 api key and set it as an environment variable (`AI21_API_KEY`)

## Wrappers | ラッパー

### LLM | LLM

AI21 LLMのラッパーが存在し、以下の方法でアクセスできます

> There exists an AI21 LLM wrapper, which you can access with

```python
from langchain.llms import AI21
```
