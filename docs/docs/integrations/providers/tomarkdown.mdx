# 2Markdown | 2Markdown

> [2markdown](https://2markdown.com/) サービスは、ウェブサイトのコンテンツを構造化されたマークダウンファイルに変換します。
>
> > [2markdown](https://2markdown.com/) service transforms website content into structured markdown files.

## Installation and Setup | インストールとセットアップ

`API key`が必要です。入手方法については[こちらの手順](https://2markdown.com/login)をご覧ください。

> We need the `API key`. See [instructions how to get it](https://2markdown.com/login).
