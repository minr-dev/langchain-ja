# Postgres Embedding | Postgres の埋め込み

> [pg\_embedding](https://github.com/neondatabase/pg_embedding)は、`Postgres`と`Hierarchical Navigable Small Worlds`アルゴリズムを使用してベクトル類似性検索を行うためのオープンソースパッケージです。このアルゴリズムは、近似最近傍探索に利用されます。
>
> > [pg\_embedding](https://github.com/neondatabase/pg_embedding) is an open-source package for
> > vector similarity search using `Postgres` and the `Hierarchical Navigable Small Worlds`
> > algorithm for approximate nearest neighbor search.

## Installation and Setup | インストールとセットアップ

いくつかのPythonパッケージをインストールする必要があります。

> We need to install several python packages.

```bash
pip install openai
pip install psycopg2-binary
pip install tiktoken
```

[使用例](/docs/integrations/vectorstores/pgembedding)をご覧ください。

from langchain.vectorstores import PGEmbedding
