# Salute Devices | 敬礼デバイス

Salute DevicesはGigaChat LLMのモデルを提供しています。

> Salute Devices provides GigaChat LLM's models.

GigaChatへのアクセス方法の詳細については、[こちらをご覧ください](https://developers.sber.ru/docs/ru/gigachat/api/integration)。

> For more info how to get access to GigaChat [follow here](https://developers.sber.ru/docs/ru/gigachat/api/integration).

## Installation and Setup | インストールとセットアップ

GigaChatパッケージは、PyPIからpipを使用してインストールできます：

> GigaChat package can be installed via pip from PyPI:

```bash
pip install gigachat
```

## LLMs | LLMs

[使用例](/docs/integrations/llms/gigachat)をご覧ください。

> See a [usage example](/docs/integrations/llms/gigachat).

```python
from langchain.llms import GigaChat
```

## Chat models | チャットモデル

[使用例](/docs/integrations/chat/gigachat)をご覧ください。

> See a [usage example](/docs/integrations/chat/gigachat).

```python
from langchain.chat_models import GigaChat
```
