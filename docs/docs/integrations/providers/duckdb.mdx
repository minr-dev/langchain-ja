# DuckDB | DuckDB

> [DuckDB](https://duckdb.org/)は、プロセス内で動作するSQL OLAPデータベース管理システムです。
>
> > [DuckDB](https://duckdb.org/) is an in-process SQL OLAP database management system.

## Installation and Setup | インストールとセットアップ

まず、`duckdb` Pythonパッケージをインストールする必要があります。

> First, you need to install `duckdb` python package.

```bash
pip install duckdb
```

## Document Loader | ドキュメントローダー

[使用例](/docs/integrations/document_loaders/duckdb)をご覧ください。

> See a [usage example](/docs/integrations/document_loaders/duckdb).

```python
from langchain.document_loaders import DuckDBLoader
```
