# Twitter | Twitter

> [Twitter](https://twitter.com/)は、オンラインのソーシャルメディアおよびソーシャルネットワーキングサービスです。
>
> > [Twitter](https://twitter.com/) is an online social media and social networking service.

## Installation and Setup | インストールとセットアップ

```bash
pip install tweepy
```

ローダーを初期化するには、`Twitter API` トークンを用い、またTwitterの`username`を設定する必要があります。

> We must initialize the loader with the `Twitter API` token, and we need to set up the Twitter `username`.
