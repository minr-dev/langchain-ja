# Ollama | オラマ

> [Ollama](https://ollama.ai/)はPythonライブラリです。これにより、LLaMA2のようなオープンソースの大規模言語モデルをローカルで実行することができます。
>
> > [Ollama](https://ollama.ai/) is a python library. It allows you to run open-source large language models,
> > such as LLaMA2, locally.
>
> `Ollama`は、モデルの重み、設定、データをModelfileによって定義された単一のパッケージにまとめます。GPUの使用を含むセットアップと設定の詳細を最適化します。サポートされているモデルとそのバリアントの完全なリストについては、[Ollamaモデルライブラリ](https://ollama.ai/library)をご覧ください。
>
> > `Ollama` bundles model weights, configuration, and data into a single package, defined by a Modelfile.
> > It optimizes setup and configuration details, including GPU usage.
> > For a complete list of supported models and model variants, see the [Ollama model library](https://ollama.ai/library).

`Ollama`をLangChainで使用する方法の詳細については、[このガイド](https://python.langchain.com/docs/guides/local_llms#quickstart)をご覧ください。

> See [this guide](https://python.langchain.com/docs/guides/local_llms#quickstart) for more details
> on how to use `Ollama` with LangChain.

## Installation and Setup | インストールとセットアップ

[これらの指示](https://github.com/jmorganca/ollama?tab=readme-ov-file#ollama)に従って、ローカルのOllamaインスタンスを設定し、実行してください。使用するためには、環境変数`ANYSCALE_API_BASE`と`ANYSCALE_API_KEY`を設定する必要があります。

> Follow [these instructions](https://github.com/jmorganca/ollama?tab=readme-ov-file#ollama)
> to set up and run a local Ollama instance.
> To use, you should set up the environment variables `ANYSCALE_API_BASE` and
> `ANYSCALE_API_KEY`.
