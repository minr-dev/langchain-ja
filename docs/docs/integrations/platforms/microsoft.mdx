### Azure Blob Storage | Azure Blob Storage

> [Azure Blob Storage](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blobs-introduction)は、クラウド用のMicrosoftのオブジェクトストレージソリューションです。Blob Storageは、大量の非構造化データの保存に最適化されています。非構造化データとは、特定のデータモデルや定義に従わないデータで、テキストやバイナリデータなどが含まれます。
>
> > [Azure Blob Storage](https://learn.microsoft.com/en-us/azure/storage/blobs/storage-blobs-introduction) is Microsoft's object storage solution for the cloud. Blob Storage is optimized for storing massive amounts of unstructured data. Unstructured data is data that doesn't adhere to a particular data model or definition, such as text or binary data.

> [Azure Files](https://learn.microsoft.com/en-us/azure/storage/files/storage-files-introduction)は、業界標準のServer Message Block（`SMB`）プロトコル、Network File System（`NFS`）プロトコル、および`Azure Files REST API`を介してアクセス可能な、クラウド内で完全に管理されたファイル共有を提供します。`Azure Files`は`Azure Blob Storage`を基盤としています。
>
> > [Azure Files](https://learn.microsoft.com/en-us/azure/storage/files/storage-files-introduction) offers fully managed
> > file shares in the cloud that are accessible via the industry standard Server Message Block (`SMB`) protocol,
> > Network File System (`NFS`) protocol, and `Azure Files REST API`. `Azure Files` are based on the `Azure Blob Storage`.

`Azure Blob Storage`は以下の目的で設計されています：

> `Azure Blob Storage` is designed for:

* 画像やドキュメントを直接ブラウザに提供する。
  > Serving images or documents directly to a browser.
* 分散アクセスのためのファイル保存。
  > Storing files for distributed access.
* ストリーミングビデオとオーディオ。
  > Streaming video and audio.
* ログファイルへの書き込み。
  > Writing to log files.
* バックアップとリストア、災害復旧、およびアーカイブのためのデータ保存。
  > Storing data for backup and restore, disaster recovery, and archiving.
* オンプレミスまたはAzureでホストされるサービスによる分析のためのデータ保存。
  > Storing data for analysis by an on-premises or Azure-hosted service.

> [Microsoft Excel](https://en.wikipedia.org/wiki/Microsoft_Excel)は、Windows、macOS、Android、iOS、iPadOS用にMicrosoftが開発したスプレッドシートエディタです。計算機能、グラフ作成ツール、ピボットテーブル、Visual Basic for Applications（VBA）というマクロプログラミング言語を搭載しています。Excelは、Microsoft 365スイートソフトウェアの一部です。

`UnstructuredExcelLoader`は、`Microsoft Excel`ファイルを読み込むために使用されます。このローダーは`.xlsx`ファイルと`.xls`ファイルの両方に対応しています。ページの内容はExcelファイルの生のテキストになります。ローダーを`"elements"`モードで使用する場合、ExcelファイルのHTML表現がドキュメントメタデータ内の`text_as_html`キーにて利用可能になります。

> The `UnstructuredExcelLoader` is used to load `Microsoft Excel` files. The loader works with both `.xlsx` and `.xls` files.
> The page content will be the raw text of the Excel file. If you use the loader in `"elements"` mode, an HTML
