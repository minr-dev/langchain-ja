# Figma | Figma

> [Figma](https://www.figma.com/)はインターフェースデザインのための協働型ウェブアプリケーションです。
>
> > [Figma](https://www.figma.com/) is a collaborative web application for interface design.

## Installation and Setup | インストールとセットアップ

Figma APIは`access token`、`node_ids`、および`file key`を必要とします。

> The Figma API requires an `access token`, `node_ids`, and a `file key`.

`file key`はURLから取得できます。https://www.figma.com/file/{filekey}/sampleFilename

> The `file key` can be pulled from the URL.  https://www.figma.com/file/{filekey}/sampleFilename

`Node IDs`はURLにも含まれています。何かをクリックして、'?node-id={node\_id}'パラメータを探してください。

> `Node IDs` are also available in the URL. Click on anything and look for the '?node-id={node\_id}' param.

`アクセストークン`については、[こちらの手順](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens)をご覧ください。

> `Access token` [instructions](https://help.figma.com/hc/en-us/articles/8085703771159-Manage-personal-access-tokens).
