# Weather | 天気

> [OpenWeatherMap](https://openweathermap.org/)は、オープンソースの天気情報サービスプロバイダーです。
>
> > [OpenWeatherMap](https://openweathermap.org/) is an open-source weather service provider.

## Installation and Setup | インストールとセットアップ

```bash
pip install pyowm
```

`OpenWeatherMap API token`を設定する必要があります。

> We must set up the `OpenWeatherMap API token`.
