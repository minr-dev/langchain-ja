# Hugging Face | Hugging Face

[Hugging Face Platform](https://huggingface.co/)に関連するすべての機能。

> All functionality related to the [Hugging Face Platform](https://huggingface.co/).

## LLMs | LLMs

### Hugging Face Hub | Hugging Face Hub

> [Hugging Face Hub](https://huggingface.co/docs/hub/index)は、35万以上のモデル、7.5万のデータセット、15万のデモアプリ（Spaces）を含む、すべてがオープンソースで公開されているオンラインプラットフォームです。人々が簡単に協力し、共に機械学習を構築できる場所として機能しています。Hubは、誰もが探索、実験、協力し、機械学習を用いた技術を構築できる中心的な場所として活動しています。
>
> > The [Hugging Face Hub](https://huggingface.co/docs/hub/index) is a platform
> > with over 350k models, 75k datasets, and 150k demo apps (Spaces), all open source
> > and publicly available, in an online platform where people can easily
> > collaborate and build ML together. The Hub works as a central place where anyone
> > can explore, experiment, collaborate, and build technology with Machine Learning.

使用するには、`huggingface_hub` Python [パッケージをインストール](https://huggingface.co/docs/huggingface_hub/installation)しておく必要があります。

> To use, we should have the `huggingface_hub` python [package installed](https://huggingface.co/docs/huggingface_hub/installation).

```bash
pip install huggingface_hub
```

[使用例](/docs/integrations/llms/huggingface_hub)をご覧ください。

> See a [usage example](/docs/integrations/llms/huggingface_hub).

```python
from langchain.llms import HuggingFaceHub
```

### Hugging Face Local Pipelines | Hugging Face ローカルパイプライン

Hugging Faceのモデルは、`HuggingFacePipeline` クラスを通じてローカルで実行することができます。

> Hugging Face models can be run locally through the `HuggingFacePipeline` class.

`transformers` Pythonパッケージをインストールする必要があります。

> We need to install `transformers` python package.

```bash
pip install transformers
```

[使用例](/docs/integrations/llms/huggingface_pipelines)をご覧ください。

> See a [usage example](/docs/integrations/llms/huggingface_pipelines).

```python
from langchain.llms.huggingface_pipeline import HuggingFacePipeline
```

### Hugging Face TextGen Inference | Hugging Face TextGen 推論

> [Text Generation Inference](https://github.com/huggingface/text-generation-inference)は、テキスト生成の推論のためのRust、Python、gRPCサーバーです。[HuggingFace](https://huggingface.co/)では、LLMのapi-inferenceウィジェットを動かすために本番環境で使用されています。
>
> > [Text Generation Inference](https://github.com/huggingface/text-generation-inference) is
> > a Rust, Python and gRPC server for text generation inference. Used in production at
> > [HuggingFace](https://huggingface.co/) to power LLMs api-inference widgets.

`text_generation` Pythonパッケージをインストールする必要があります。

> We need to install `text_generation` python package.

```bash
pip install text_generation
```

[使用例](/docs/integrations/llms/huggingface_textgen_inference)をご覧ください。

> See a [usage example](/docs/integrations/llms/huggingface_textgen_inference).

```python
from langchain.llms import HuggingFaceTextGenInference
```

## Tools | ツール

### Hugging Face Hub Tools | Hugging Face Hubツール

> [Hugging Face Tools](https://huggingface.co/docs/transformers/v4.29.0/en/custom_tools)はテキストの入出力をサポートしており、`load_huggingface_tool`関数を使用してロードされます。
>
> > [Hugging Face Tools](https://huggingface.co/docs/transformers/v4.29.0/en/custom_tools)
> > support text I/O and are loaded using the `load_huggingface_tool` function.

いくつかのPythonパッケージをインストールする必要があります。

> We need to install several python packages.

```bash
pip install transformers huggingface_hub
```

[使用例](/docs/integrations/tools/huggingface_tools)をご覧ください。

> See a [usage example](/docs/integrations/tools/huggingface_tools).

```python
from langchain.agents import load_huggingface_tool
```
