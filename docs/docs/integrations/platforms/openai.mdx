## Installation and Setup | インストールとセットアップ

* Python SDKをインストールするには
  > Install the Python SDK with

```bash
pip install openai
```

* OpenAIのAPIキーを取得し、環境変数（`OPENAI_API_KEY`）として設定してください
  > Get an OpenAI api key and set it as an environment variable (`OPENAI_API_KEY`)
* OpenAIのトークナイザーを使用したい場合（Python 3.9以上でのみ利用可能）、インストールしてください
  > If you want to use OpenAI's tokenizer (only available for Python 3.9+), install it

```bash
pip install tiktoken
```

## Tokenizer | トークナイザー

`tiktoken` トークナイザーは、いくつかの場面で使用できます。デフォルトでは、OpenAIのLLM（Large Language Models）のトークン数をカウントするために使用されます。

> There are several places you can use the `tiktoken` tokenizer. By default, it is used to count tokens
> for OpenAI LLMs.

また、ドキュメントを分割する際にトークンを数えるためにも使用できます。

> You can also use it to count tokens when splitting documents with

```python
from langchain.text_splitter import CharacterTextSplitter
CharacterTextSplitter.from_tiktoken_encoder(...)
```

これについてのより詳細なウォークスルーは、[このノートブック](/docs/modules/data_connection/document_transformers/text_splitters/split_by_token#tiktoken)をご覧ください。

> For a more detailed walkthrough of this, see [this notebook](/docs/modules/data_connection/document_transformers/text_splitters/split_by_token#tiktoken)
