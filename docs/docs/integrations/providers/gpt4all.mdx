# GPT4All | GPT4All

このページでは、LangChain内での`GPT4All`ラッパーの使用方法について説明します。チュートリアルは、インストールとセットアップに続いて、例を用いた使用方法の2部構成になっています。

> This page covers how to use the `GPT4All` wrapper within LangChain. The tutorial is divided into two parts: installation and setup, followed by usage with an example.

## Installation and Setup | インストールとセットアップ

* `pip install pyllamacpp`でPythonパッケージをインストールしてください
  > Install the Python package with `pip install pyllamacpp`
* [GPT4Allモデル](https://github.com/nomic-ai/pyllamacpp#supported-model)をダウンロードして、ご希望のディレクトリに配置してください。
  > Download a [GPT4All model](https://github.com/nomic-ai/pyllamacpp#supported-model) and place it in your desired directory

## Usage | 使用方法

### GPT4All | GPT4All

GPT4Allラッパーを使用するには、事前にトレーニングされたモデルファイルへのパスとモデルの構成を提供する必要があります。

> To use the GPT4All wrapper, you need to provide the path to the pre-trained model file and the model's configuration.

```python
from langchain.llms import GPT4All

# Instantiate the model. Callbacks support token-wise streaming
model = GPT4All(model="./models/gpt4all-model.bin", n_ctx=512, n_threads=8)

# Generate text
response = model("Once upon a time, ")
```

また、n\_predict、temp、top\_p、top\_kなどの生成パラメータをカスタマイズすることもできます。

> You can also customize the generation parameters, such as n\_predict, temp, top\_p, top\_k, and others.

モデルの予測をストリーミングするには、CallbackManagerを追加してください。

> To stream the model's predictions, add in a CallbackManager.

```python
from langchain.llms import GPT4All
from langchain.callbacks.streaming_stdout import StreamingStdOutCallbackHandler

# There are many CallbackHandlers supported, such as
# from langchain.callbacks.streamlit import StreamlitCallbackHandler

callbacks = [StreamingStdOutCallbackHandler()]
model = GPT4All(model="./models/gpt4all-model.bin", n_ctx=512, n_threads=8)

# Generate text. Tokens are streamed through the callback manager.
model("Once upon a time, ", callbacks=callbacks)
```
