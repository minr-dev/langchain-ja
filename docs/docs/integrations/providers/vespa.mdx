# Vespa | Vespa

> [Vespa](https://vespa.ai/)は、フル機能を備えた検索エンジンおよびベクターデータベースです。ベクター検索（ANN）、レキシカル検索、構造化データにおける検索を、同一のクエリでサポートしています。
>
> > [Vespa](https://vespa.ai/) is a fully featured search engine and vector database.
> > It supports vector search (ANN), lexical search, and search in structured data, all in the same query.

## Installation and Setup | インストールとセットアップ

```bash
pip install pyvespa
```
