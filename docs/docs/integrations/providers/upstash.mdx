# Upstash Redis | Upstash Redis

Upstashは、開発者がスケールアップ時のデータベース運用の複雑さを心配せずに、強力なアプリケーションを構築できるように、サーバーレスデータベースやメッセージングプラットフォームを提供しています。

> Upstash offers developers serverless databases and messaging platforms to build powerful applications without having to worry about the operational complexity of running databases at scale.

このページでは、LangChainで[Upstash Redis](https://upstash.com/redis)を使用する方法について説明します。

> This page covers how to use [Upstash Redis](https://upstash.com/redis) with LangChain.

## Installation and Setup | インストールとセットアップ

* `pip install upstash-redis` コマンドで Upstash Redis Python SDK をインストールできます。
  > Upstash Redis Python SDK can be installed with `pip install upstash-redis`
* [Upstashコンソール](https://console.upstash.com)で、グローバルに分散され、低遅延で高可用性のデータベースを作成することができます。
  > A globally distributed, low-latency and highly available database can be created at the [Upstash Console](https://console.upstash.com)

## Integrations | 統合

Upstash-LangChainのすべての統合は、LangChainのラッパーとして利用される`upstash-redis` Python SDKに基づいています。このSDKは、コンソールから提供されるUPSTASH\_REDIS\_REST\_URLおよびUPSTASH\_REDIS\_REST\_TOKENパラメータを使用してUpstash Redis DBにアクセスします。このSDKの大きな利点の一つは、REST APIを使用していることです。これにより、TCP接続をサポートしていないサーバーレスプラットフォームやエッジなど、任意のプラットフォームで実行することが可能です。

> All of Upstash-LangChain integrations are based on `upstash-redis` Python SDK being utilized as wrappers for LangChain.
> This SDK utilizes Upstash Redis DB by giving UPSTASH\_REDIS\_REST\_URL and UPSTASH\_REDIS\_REST\_TOKEN parameters from the console.
> One significant advantage of this is that, this SDK uses a REST API. This means, you can run this in serverless platforms, edge or any platform that does not support TCP connections.

### Cache | キャッシュ

[Upstash Redis](https://upstash.com/redis)は、LLMのプロンプトとレスポンスのキャッシュとして使用できます。

> [Upstash Redis](https://upstash.com/redis) can be used as a cache for LLM prompts and responses.

このキャッシュをインポートするには：

> To import this cache:

```python
from langchain.cache import UpstashRedisCache
```

LLMを使用するには：

> To use with your LLMs:

```python
import langchain
from upstash_redis import Redis

URL = "<UPSTASH_REDIS_REST_URL>"
TOKEN = "<UPSTASH_REDIS_REST_TOKEN>"

langchain.llm_cache = UpstashRedisCache(redis_=Redis(url=URL, token=TOKEN))
```
