# Helicone | Helicone

このページでは、LangChain内で[Helicone](https://helicone.ai)エコシステムを使用する方法について説明します。

> This page covers how to use the [Helicone](https://helicone.ai) ecosystem within LangChain.

## What is Helicone? | Heliconeとは何ですか？

Heliconeは、あなたのOpenAIトラフィックをプロキシし、支出、レイテンシ、使用状況に関する重要な洞察を提供する[オープンソース](https://github.com/Helicone/helicone)の可観測性プラットフォームです。

> Helicone is an [open-source](https://github.com/Helicone/helicone) observability platform that proxies your OpenAI traffic and provides you key insights into your spend, latency and usage.

![Helicone](/img/HeliconeDashboard.png)

## Quick start | クイックスタート

LangChain環境を使用する際には、次のパラメータを追加するだけです。

> With your LangChain environment you can just add the following parameter.

```bash
export OPENAI_API_BASE="https://oai.hconeai.com/v1"
```

次に、[helicone.ai](https://helicone.ai/onboarding?step=2) にアクセスしてアカウントを作成し、ダッシュボードにあなたのOpenAI APIキーを追加してログを閲覧してください。

> Now head over to [helicone.ai](https://helicone.ai/onboarding?step=2) to create your account, and add your OpenAI API key within our dashboard to view your logs.

![Helicone](/img/HeliconeKeys.png)

## How to enable Helicone caching | Heliconeキャッシングを有効にする方法

```python
from langchain.llms import OpenAI
import openai
openai.api_base = "https://oai.hconeai.com/v1"

llm = OpenAI(temperature=0.9, headers={"Helicone-Cache-Enabled": "true"})
text = "What is a helicone?"
print(llm(text))
```

[Helicone キャッシングのドキュメント](https://docs.helicone.ai/advanced-usage/caching)

> [Helicone caching docs](https://docs.helicone.ai/advanced-usage/caching)

## How to use Helicone custom properties | Heliconeカスタムプロパティの使用方法

```python
from langchain.llms import OpenAI
import openai
openai.api_base = "https://oai.hconeai.com/v1"

llm = OpenAI(temperature=0.9, headers={
        "Helicone-Property-Session": "24",
        "Helicone-Property-Conversation": "support_issue_2",
        "Helicone-Property-App": "mobile",
      })
text = "What is a helicone?"
print(llm(text))
```

[Helicone プロパティのドキュメント](https://docs.helicone.ai/advanced-usage/custom-properties)

> [Helicone property docs](https://docs.helicone.ai/advanced-usage/custom-properties)
