# Tair | Tair

このページでは、LangChain内でTairエコシステムを使用する方法について説明しています。

> This page covers how to use the Tair ecosystem within LangChain.

## Installation and Setup | インストールとセットアップ

`pip install tair`を使用してTair Python SDKをインストールしてください。

> Install Tair Python SDK with `pip install tair`.

## Wrappers | ラッパー

### VectorStore | VectorStore

TairVectorの周りにはラッパーが存在し、それをベクトルストアとして、セマンティック検索や例示選択に使用することができます。

> There exists a wrapper around TairVector, allowing you to use it as a vectorstore,
> whether for semantic search or example selection.

このベクトルストアをインポートするには：

> To import this vectorstore:

```python
from langchain.vectorstores import Tair
```

Tairラッパーのより詳細なウォークスルーについては、[このノートブック](/docs/integrations/vectorstores/tair)をご覧ください。

> For a more detailed walkthrough of the Tair wrapper, see [this notebook](/docs/integrations/vectorstores/tair)
